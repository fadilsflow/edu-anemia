# Edu Anemia

Web application for anemia education and risk assessment.

## Features

- Educational resources about anemia
- Risk assessment calculator based on WHO guidelines
- Interactive quiz to test knowledge

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS

## Getting Started

1. Install dependencies:

```bash
bun install
```

2. Run the development server:

```bash
bun run dev
```

3. Open http://localhost:3000
